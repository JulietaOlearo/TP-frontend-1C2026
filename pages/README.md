# Perfumes VS

## Descripción
Sitio web e-commerce de perfumes desarrollado con HTML, CSS y JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES6
- LocalStorage
- GitHub Pages

## Funcionalidades

- Visualización dinámica de productos mediante JSON.
- Carga de productos utilizando Fetch.
- Carrito de compras.
- Agregar productos.
- Eliminar productos.
- Vaciar carrito.
- Persistencia de datos con LocalStorage.

## Estructura del proyecto

- css: estilos del sitio.
- js: lógica e interacción.
- data: información de productos.
- img: recursos gráficos.
- pages: páginas secundarias.
